# AI Demo 项目

这是一个基于 C++ 的 AI 演示项目，使用 CMake 构建系统。

## 项目结构

```
ai-demo/
├── CMakeLists.txt
├── main.cpp
└── build/  # 构建目录
```

## 构建说明

1. 确保已安装 CMake 和 C++ 编译器
2. 创建构建目录并配置项目：
   ```bash
   mkdir build
   cd build
   cmake ..
   ```
3. 编译项目：
   ```bash
   cmake --build .
   ```
4. 运行生成的可执行文件：
   ```bash
   ./cpp_server
   ```
